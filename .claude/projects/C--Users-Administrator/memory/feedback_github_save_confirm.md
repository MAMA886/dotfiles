每次写完程序/脚本/配置文件后，先问用户"要不要存到 GitHub？"，用户确认后再 push。

**Why:** 用户不想自动推送，要每次手动确认。
**How to apply:** 完成代码编写后，先展示结果，再询问是否需要 push 到 GitHub。
