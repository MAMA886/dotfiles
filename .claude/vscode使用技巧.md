# VS Code 使用技巧

## 常用快捷键

| 操作 | 快捷键 |
|------|--------|
| 命令面板 | Ctrl+Shift+P |
| 打开终端 | Ctrl+` |
| 新建文件 | Ctrl+N |
| 保存文件 | Ctrl+S |
| 搜索文件 | Ctrl+P |
| 全局搜索 | Ctrl+Shift+F |
| 多光标 | Alt+点击 |
| 整行移动 | Alt+↑/↓ |
| 复制整行 | Shift+Alt+↑/↓ |
| 格式化代码 | Shift+Alt+F |
| 折叠代码 | Ctrl+Shift+[ |
| 展开代码 | Ctrl+Shift+] |

## 小圆（Claude Code）相关

- **打开小圆**：Ctrl+Shift+P → 输入 "Claude Code: Chat"
- **新窗口聊天**：桌面双击"和小圆聊天"快捷方式（独立窗口，不影响当前对话）
- **切换对话**：Ctrl+Shift+P → "Claude Code: Switch Session"

## 写代码常用

- **快速注释**：选中代码 → Ctrl+/（单行）或 Shift+Alt+A（多行）
- **重命名变量**：选中 → F2（自动修改所有引用）
- **跳转到定义**：F12 或 Ctrl+点击
- **返回上一位置**：Alt+←
- **预览 Markdown**：Ctrl+Shift+V
- **禅模式**：Ctrl+K Z（全屏无干扰）

## 终端技巧

- 终端里按 ↑ 键回看历史命令
- Ctrl+C 终止当前命令
- 拖拽文件到终端 = 自动粘贴文件路径
- 选中文字右键 = 复制（终端里 Ctrl+C 是终止，不是复制）

## 窗口管理

- Ctrl+\ 分屏编辑
- Ctrl+1/2/3 切换分屏焦点
- Ctrl+B 开关侧边栏
- Ctrl+Shift+E 文件资源管理器
